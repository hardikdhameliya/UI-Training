Validation carried out by Javascript are called as client side validation.
Because they are executed by web browsers on client machine.
Consequence of this is, client has all the control of deciding whether JS is enabled or disabled. 
If JS is disabled by client, JS validation won't take place.
Hence, as a application developer you must always have server side validation. 
Server side validations is important, mandate.  
Client side validation will reduce load on server, network calls.

Use let keyword instead of var keyword when declaring variable.
